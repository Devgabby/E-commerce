Assignment #1

Web System Development

Design a web-based application which can perform arithmetic and logical computation via Graphical User Interface (GUL)

GUI requirement

- tools like HTML, CSS (optional) can be used to design the graphical components for the application.

- the interface should be interactive i.e. user can give input and get computed result.

- the arrangement of component, design, color and other apperent.

- to reduce complexity to GUI, the number of components can be grouped together according to their similarity of split into multiple levels (linked screens)

Funtional Requirement

- The application support two or more

- Operands must be computed and compared

  • Basic Arithmetic operators (e.g. addition, subtraction, etc)

  • Advanced Arithmetic operators (e.g. Square root, Trigonometric, etc)

  • Rational Operators in simple expressions

  • Rational and Logical operators in complex expressions

    • Note: At least (5) number of operators must be included from each category of operators.
